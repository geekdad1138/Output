Starting tomorrow June 1st, GitHub is moving to a new usage based pricing plan based on “GitHub AI Credits”.

The cited reason for this change is to align the prices GitHub (aka Microsoft) is paying to what the user is paying.

From the article, Chief Product Officer [Mario Rodriguez](https://www.linkedin.com/in/mariorodriguez3/) writes:
“GitHub has absorbed much of the escalating inference cost”

So if you were a monthly user paying $39 to GitHub, and GitHub’s model calls to a provider like Anthropic cost them $10,000 then Microsoft would lose out.

This is following similar moves by IDEs like Cursor, trying to balance keeping users on their platform versus going bankrupt on token cost.

It will be interesting to see the effects on the industry, if users roll back their token usage, which could greatly reduce the revenue coming into the AI model providers. Or if companies just pay that bill, and the challenge grows for how to make a profit using AI.
